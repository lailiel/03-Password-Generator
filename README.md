# 03-Challenge
Repository for coding bootcamp 03 challenge, javascript.

When clicking with the "generate password" user is prompted for password length.
After entering an accepted length (between 8 and 128 characters),
user will then be prompted to select whether to include lowercase, uppercase, numbers, and special characters in the generated passwords.

When promts are completed, password is displayed in the window.
